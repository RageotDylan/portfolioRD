Memory

Ce projet est un jeu de mémoire interactif. L'objectif du jeu est de retourner des paires de cartes pour retrouver des symboles identiques. Voici les principales fonctionnalités et caractéristiques :

Interface Utilisateur :
Une interface conviviale avec des cartes disposées en grille, chaque carte ayant un symbole caché derrière elle.

Chronomètre :
Un chronomètre qui suit le temps écoulé en minutes et secondes, permettant aux joueurs de voir combien de temps ils ont mis pour terminer le jeu.

Compteur de Coups :
Un compteur qui enregistre le nombre de clics effectués par le joueur, offrant un retour sur la performance.

Vérification des Paires :
Une logique intégrée pour vérifier si deux cartes retournées correspondent. Si elles correspondent, elles restent visibles ; sinon, elles sont retournées après un court délai.

Verrouillage du Plateau :
Un mécanisme qui empêche les clics supplémentaires pendant le traitement des cartes, garantissant une expérience de jeu fluide.

Réinitialisation du Jeu :
Une fonctionnalité permettant de réinitialiser le jeu à tout moment, remettant à zéro le chronomètre et le compteur de coups.
